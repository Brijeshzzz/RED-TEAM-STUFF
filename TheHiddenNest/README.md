# The Hidden Nest – CTF Challenge

## Description
Everything you need is already inside the file.

## Difficulty
Beginner to Intermediate (1–3 hours)

## Build Instructions
Run the following commands:

```bash
chmod +x create_challenge.sh
./create_challenge.sh
```

The final challenge file will be generated at:

dist/welcome.jpg

## Required Tools
- ExifTool
- Binwalk
- Strings
- CyberChef
- Audacity or Morse Decoder
